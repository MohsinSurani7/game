# Confusing Maze Game (Android)

Yeh ek Android platformer prototype hai jisme levels intentionally confusing banaye gaye hain:

- Fake platforms jo touch karte hi toot jaate hain.
- Hidden platforms jo player ke paas aane par reveal hoti hain.
- Deadly trap blocks jo normal path ke side me achanak aate hain.
- Neeche se "death fog" rise karti hai jo player ko pressure me rakhti hai.
- Random reverse-gravity pulse jo gameplay ko unpredictable banata hai.

## Run karne ka tareeqa

1. Android Studio me is folder ko open karein.
2. Gradle sync hone dein.
3. Emulator/device select karke app run karein.

## Controls

- Screen left half: left move
- Screen right half: right move
- Screen upper half par tap: jump

Har level complete karte hi next level aur zyada difficult generate hota hai.
